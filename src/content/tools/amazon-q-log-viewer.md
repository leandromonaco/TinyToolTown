A tiny debug logs viewer for Amazon Q inside Visual Studio Code.

It shows what’s usually hidden — requests, responses, tool calls, errors, and even the underlying LLM interactions — so you can see what the AI is actually doing.

Built it because GitHub Copilot has this (Debug View)… and Amazon Q didn’t.
Made debugging and demos unnecessarily painful.

It’s delightful because it turns a black box into something you can poke at.
Click → inspect → “ohhh, that’s what the model saw.”
