Bingscape is a lightweight Tauri desktop app that fetches and sets the daily Bing image as your macOS wallpaper. It supports automatic hourly updates, manual sync, multi-display setups, and background operation via the system tray. With persistent settings and optional start at login, Bingscape makes keeping your desktop fresh effortless. Designed specifically for macOS, it offers a clean and intuitive user experience.
