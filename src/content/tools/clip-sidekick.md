Today's information workers live in a constant loop: copy text, switch to an AI chat, paste, wait, copy the answer, switch back. Clip Sidekick breaks that cycle. Copy text in any app and a notification bubble appears with AI tasks at your fingertips. Pick one and the result lands right back where you started, whether proofread, rewritten, summarized, or transformed, all without leaving the app you're working in. Clipboard history, bookmarks, and a complete AI editing toolkit come together in one small, hotkey driven tool designed to keep you in the flow. It also supports MCP tools and agent Skills. Use the built in GitHub models from the GitHub Copilot CLI or bring your own, whether cloud hosted or running locally.
