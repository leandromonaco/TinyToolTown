Browse and manage your GitHub Copilot CLI config in a fast, filterable TUI. 11 tabs cover plugins, MCP servers, skills, agents, and settings. Toggle, upgrade, and remove plugins or MCP servers directly from the dashboard. Includes a doctor command for MCP server health probes.
