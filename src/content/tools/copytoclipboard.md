A simple html page, that you save locally and open locally, allows to save text and have it in the clipboard in one click. (has to be local, or hosted on https to push into clipboard)  

I built it because I always needed to type in the same information over and over, I wanted to have a one stop shop, to save text, and get it back in the clipboard in ONE click.

It brings consistency into my commit messages, i can get my long email address and paste it quickly, (no typo) I can re-use some text formulation in documentations and jira tickets.  Also, sice it is local, it's ok with company policies as it won't request access external sites.
