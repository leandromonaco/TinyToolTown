Dev Days Giveaway is a flexible, static site for running raffles at events. It supports both Supabase-backed and local file modes, making it easy to deploy online or run offline. Features include a public entry form, animated winner draws, admin dashboard, and one-win-per-person enforcement. Perfect for lightweight, secure event raffles with minimal setup.
