Built out of frustration that the existing DevTools location overrides only worked for a single location at a time, this tool was born to override everything, location, speed, heading, attitude, accuracy, the whole she-bang.

It also allows you to drop GeoJSON or KML routes in and the tool will simulate travelling along that route, perfect for testing real time tracking applications.
