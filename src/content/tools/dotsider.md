dotsider lets you open any .NET DLL, EXE, or NuGet package in a terminal and immediately see everything inside — types, methods, IL bytecode, strings, PE headers, dependencies, and size breakdowns. Inspired by binsider for ELF binaries, it brings that same quick-glance workflow to .NET. Vim-style keybindings, so you never leave your terminal to answer "what's actually in this assembly?"
