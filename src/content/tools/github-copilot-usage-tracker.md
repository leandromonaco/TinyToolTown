GitHub Copilot Usage Tracker is a cross-platform system tray app that visualizes your Copilot Premium request usage in real time. It features secure authentication via GitHub OAuth or manual token entry, and auto-refreshes your usage data every five minutes. The app is easy to set up and works seamlessly on macOS, Windows, and Linux. Stay on top of your Copilot usage with clear, interactive progress bars.
