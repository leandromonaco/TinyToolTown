Graph X-Ray is a free, open source browser extension that reveals Microsoft Graph API calls made by Microsoft admin portals as you use them. It captures requests, shows the details in browser developer tools, and can generate ready-to-use code snippets in PowerShell, Python, C#, JavaScript, Go, Java, and Objective-C. It is delightful because it gives admins and developers X-ray vision into portals and turns manual clicks into automation scripts.
