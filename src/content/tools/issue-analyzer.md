A Next.js app that analyzes GitHub issues using multiple LLM models in parallel via the [GitHub Copilot SDK](https://github.com/github/copilot-sdk). Each model posts its analysis as a separate GitHub comment, giving you diverse perspectives on bugs, feature requests, PRDs, or any issue content.
