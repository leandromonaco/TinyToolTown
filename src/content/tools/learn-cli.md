A cross-platform command-line learning app that integrates with Microsoft Learn documentation via MCP (Model Context Protocol). Normally you need an AI agent and language model to interact with an MCP server. In this case we are breaking dynamic discovery and grounding rules, and just use the MCP server as API. Enter a simple search string, and get a response.
