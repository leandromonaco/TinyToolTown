ms-todo-cli is a small TypeScript command-line client for Microsoft To Do. It lets you log in with Microsoft device code flow, manage lists, tasks, and checklist steps, and script everything through clean JSON output. Built for people who want their task manager to cooperate with the terminal instead of trapping their data inside a GUI.
