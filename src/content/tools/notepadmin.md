A minimal, cross-platform plain-text editor — a marriage of classic Windows Notepad (text only, no formatting!) and Linux gedit. Really just an app around a simple textbox that can open and save text files. Did I already say there is no formatting (which is why I used to love classic Notepad). Built with C#, .NET 10, and Avalonia UI, using GitHub Copilot CLI on Ubuntu 24.04 on WSL2 on Windows 11. Should run on Linux and Windows. Feedback? Create an issue 🙏
