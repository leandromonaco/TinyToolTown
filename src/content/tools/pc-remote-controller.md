A Windows-focused Flask app that lets you control your PC from a phone browser on the same local network.
The app provides:

- PIN lock screen
- Mouse control (move, left/right/middle click, scroll, tap-to-click)
- Arrow keys and quick hotkeys (Esc, F11, Windows)
- Text console with send, live-typing sync, and fast backspace repeat
- Live desktop preview (/screen_stream) with fullscreen mode
- Status cards (battery, volume, time)
- File upload to common user folders (Desktop, Downloads, Pictures, Videos)
