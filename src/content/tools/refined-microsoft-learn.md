Refined Microsoft Learn is a free, open source browser extension that makes Microsoft Learn easier to read. It removes distracting side content, expands articles and code samples to use the full browser width, and adds controls for showing or hiding clutter. It is tiny and delightful because it fixes one everyday annoyance for developers and admins who spend a lot of time reading Microsoft docs.
