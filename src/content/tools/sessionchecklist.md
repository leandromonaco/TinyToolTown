A tiny always-on-top Windows desktop widget that keeps your AI coding session habits honest. Three checkboxes — Start, During, and End of session — each with inline reminders about context management: what to @mention, when to do a handover, and how to save your session state to CONTEXT.md. Built in PowerShell + WPF, no install required. Collapses to a single header strip when you need it out of the way.
