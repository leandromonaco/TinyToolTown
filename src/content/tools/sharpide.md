SharpIDE is a modern, cross platform IDE for .NET, built with .NET & Godot.
Features include C# Project/Solution management, NuGet package management, building, running and debugging (via my SharpDbg package, also open-source!), as well as C# and Razor syntax highlighting, diagnostics, code actions, completions, Go To Definition, Find All References, symbol hover info and method signature help info.

I argue it is a tiny tool, at 94MB zipped :)
