I found it hard to find the file I'm looking for in boring, grey directory listings, so initially wrote this to colorize by extension and attribute, but it's grown far beyond that initial need with support for text mode file type icons via NerdFonts, visual tree hierarchy, and a ton of other features.  Hope you find it as useful as I do.
