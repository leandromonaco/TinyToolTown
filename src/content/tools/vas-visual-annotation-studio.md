VAS — Visual Annotation Studio is a screenshot and image annotation tool designed exclusively for macOS. Capture, annotate, and export in one seamless workflow.

CAPTURE MODES
• Full Screen — capture your entire display in one click
• Window — click any window to capture it precisely
• Region — drag to select any rectangular area
• Delayed — 1–3 second countdown to catch tooltips, menus, and other transient UI

ANNOTATION TOOLS
• Pen, line, arrow, rectangle, ellipse, and filled shapes
• Text labels with customizable font, size, color, and background
• Numbered markers with auto-increment
• Mosaic and Gaussian blur for redacting sensitive content
• Symbol stickers for quick visual markers
• Highlighter for semi-transparent emphasis

ADVANCED FEATURES
• Privacy Scan — automatically detects and redacts credit card numbers, phone numbers, and ID numbers
• QR Code Scan — select a QR code to instantly read its content
• Batch Convert — convert multiple images at once (JPG/PNG/WebP/GIF/BMP) with optional watermark and resize
• Smart Alignment — align, distribute, and center multiple objects
• Image Overlay — place external images into the editor with crop and resize

EXPORT FORMATS
JPG／PNG／WebP／GIF／BMP with customizable quality and dimensions

LANGUAGES
繁體中文, English, 日本語

Lightweight, fast, native macOS experience.
