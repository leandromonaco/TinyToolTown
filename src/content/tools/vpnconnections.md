It works as a tray icon in the task bar and shows the current state of your VPN connection. It works similar like WireGuard tray icon to connect / disconnect a connection, despite the fact, that it is not for WireGuard, but for VPN connections you added directly within windows.

This tool shows the current state of the connection in the tray, so you always know, if the connection is still there or not and it is highly customizable about what should happen if you click on the icon (left / right, one / double).
