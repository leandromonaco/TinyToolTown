VS Code MCP Install Button Generator is a free, open source web utility for creating VS Code MCP install links and badges. It supports stdio and HTTP/SSE MCP server configs, lets you customize Shields.io badge styles and colors, and generates Markdown or HTML ready for a README or project page. It is delightful because it turns the fiddly URL encoding for `vscode:mcp/install` links into a quick copy-and-paste workflow.
