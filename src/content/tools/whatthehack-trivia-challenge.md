WhatTheHack - Trivia Challenge is a simple trivia-style game built with .NET 10 and ASP.NET Core Blazor. It offers a customizable board with categories and point values, plus flows for revealing clues, showing answers, and tracking team scores. It's great for team-building, classroom activities, or as a starter template for more advanced quiz apps, with optional New Relic Browser monitoring for usage and performance insights.
