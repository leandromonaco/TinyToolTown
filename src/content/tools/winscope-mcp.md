Safe Windows diagnostics for AI agents — Sysinternals power with a baby gate.

WinScope MCP is a local .NET tool that wraps selected Microsoft Sysinternals command-line utilities behind typed, read-only commands and a local MCP server.

It gives AI agents enough diagnostic superpowers to be useful, without handing them the keys to the Windows kingdom and saying, “Try not to break anything.”
