Yako is a free, open source browser extension that replaces the default new tab page with a clean, private start page built for Microsoft users and admins. It gives quick access to 900+ Microsoft portals and blades, supports cmd.ms and MSPortals.io views, and includes useful personal touches like quick links, notes, clocks, backgrounds, dark mode, and layout customization. It is delightful because it turns a cluttered browser new tab into a fast, keyboard-friendly productivity hub with no accounts, subscriptions, or data collection.
