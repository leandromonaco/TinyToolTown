ZoomItMac is a native macOS menu bar app that brings [Sysinternals ZoomIt](https://learn.microsoft.com/en-us/sysinternals/downloads/zoomit) functionality to Mac.

I recently started using MacBook and was missing ZoomIt. I explored available options but none of the tools provided complete functionality as we get in Windows version including the new Panorama feature so I decided to Vibe code it myself.

This tiny tool, not only includes all the Windows ZoomIt features but also adds things like Redact Mode and Numbers mode where you can click on the screen and it will add a number within in red circle, good for when you want people to follow a sequence.
